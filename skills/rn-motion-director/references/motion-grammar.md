# Motion Grammar

Use this file when choosing the moving language for a topic video.

## Motion Primitives

- `draw`: lines, maps, paths, diagrams, timelines, and handwriting appear as if being traced.
- `branch`: one source splits into multiple paths, choices, species, agents, or outcomes.
- `merge`: scattered elements converge into a result, identity, product, conclusion, or decision.
- `orbit`: tokens, models, tools, planets, ideas, or actors rotate around a center to show ecosystem or dependency.
- `handoff`: one object passes energy, data, light, or responsibility to another.
- `morph`: shape, body, graph, icon, or diagram changes form to show evolution or transformation.
- `scan`: a beam, mask, cursor, or inspection layer reveals hidden structure.
- `stack`: layers accumulate, compress, reorder, or collapse to show complexity.
- `flow`: particles, text fragments, routes, money, data, people, or signals travel through a system.
- `break`: a rigid grid fractures, bends, leaks, or rewires to show disruption.
- `camera`: push, pan, orbit, crop, or depth parallax changes viewer attention.
- `material`: texture, glow, grain, blur, shadow, or edge hardness changes state.

## Topic to Metaphor Map

- history/evolution: time river, family tree, fossil layers, migration map, branching paths.
- AI/agents/software: cursor split, task graph, tool orbit, API routes, memory lattice.
- business/growth: flywheel, funnel, compounding curve, pipeline, market map.
- economics: pressure field, expanding price grid, balance scale, flow of tokens or goods.
- education/tutorial: reveal lens, assembly line, step machine, guided cursor.
- conflict/strategy: terrain map, moving front line, decision tree, resource board.
- personal productivity: clutter collapse, workflow conveyor, energy battery, calendar grid.

## Beat Construction

Each beat needs one dominant primitive and one support primitive.

Good:

`branch + draw`: family tree grows while labels appear on the branches.

`flow + handoff`: a request travels through tools and returns as a finished artifact.

`morph + material`: a rough sketch becomes a polished object while surface texture changes.

Weak:

`fade + text`: a sentence appears over a static background.

`slide + card`: a rectangular panel enters, waits, exits, and repeats.

## Repetition Rule

Reuse a primitive across the whole video to create coherence, but change what it
means. A time river can become a migration path, then a modern network. This is
better than cutting to unrelated page designs.
