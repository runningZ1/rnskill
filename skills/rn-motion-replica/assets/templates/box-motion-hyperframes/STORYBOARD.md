# Storyboard — 35s Box Motion Study

## 00.00–03.50 — Build the core chain

The technical frame and title land first. A yellow model card arrives, a small bridge node appears below it, the connector draws, then a white Codex card lands to its right.

## 03.50–07.80 — Add context

A yellow summary card slides in from the left. A compact preview card arrives at the far right, framing the graph as a video-production system without using source footage.

## 07.80–12.90 — Expand the request stack

Three compact white request/result cards appear one at a time on the right. Their elbow connectors grow from the Codex node.

## 12.90–18.50 — Morph to the routing view

The first cluster compresses and fades while a new graph replaces it: prompt card, Codex-to-model routing card, settings card, and result card. Connector lines draw after the relevant endpoints are visible.

## 18.50–30.80 — Add evidence and adapters

An evidence card arrives under the central route. Three third-party model adapters stack on the right, followed by a small local utility node and a DONE badge. Each entrance is isolated by a short hold.

## 31.20–35.00 — Push to configuration scene

The routing graph pushes left as a fresh configuration board enters from the right. Repository pill, install card, terminal card, target card, and connected status chip arrive in sequence. End on a complete, readable configuration graph.
