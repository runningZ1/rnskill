# Timing Presets

## Micro: 2.4-3.2s

Use for short vertical videos or when the hook must start almost immediately.

| Time | Beat |
| --- | --- |
| 0.00-0.20 | Pure black hold |
| 0.20-0.95 | Hero phrase types in with click SFX |
| 0.95-1.70 | Phrase holds or one short replacement types in |
| 1.70-2.50 | Final phrase hold |
| 2.50-3.20 | Smear, wipe, or cut into body |

## Standard: 4.8-5.8s

Default for tutorials, SaaS/product explainers, and cinematic AI videos.

| Time | Beat |
| --- | --- |
| 0.00-0.30 | Pure black hold |
| 0.30-1.35 | First phrase types in with click SFX |
| 1.20-2.20 | First phrase completes and holds |
| 2.20-3.55 | 1-2 replacements type in with click SFX |
| 3.55-4.70 | Final phrase types in, then holds |
| 4.70-5.80 | Transition into next scene |

## Slow: 6.0-7.5s

Use for more cinematic openings or when the next scene needs a strong setup.

| Time | Beat |
| --- | --- |
| 0.00-0.50 | Pure black hold |
| 0.50-1.90 | First phrase types in with click SFX |
| 1.60-3.00 | Phrase completes and breathes |
| 3.00-4.90 | Two word replacements type in with horizontal blur |
| 4.90-6.30 | Final phrase types in, then logo/phrase holds |
| 6.30-7.50 | White wipe or fade to body |

## Cadence

- Default: `14 chars/s`.
- Dense Chinese copy: `12 chars/s`.
- Short English terms or product names: `14-16 chars/s`.
- If a phrase cannot finish inside its typing window, shorten the copy before
  raising cadence above `16 chars/s`.
