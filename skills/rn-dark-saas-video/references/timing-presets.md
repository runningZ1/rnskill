# Timing Presets

## Sting: 8-12s

Use for logo stings, short social intros, or one feature reveal.

- 0.0-2.5s kinetic promise
- 2.5-5.0s prompt or CTA click
- 5.0-8.5s generated object / capability reveal
- 8.5-12.0s final claim

Use 3-4 scene modules.

## Standard: 30-36s

Use for Presenton-like product shorts. This is the default.

- 0.0-5.5s kinetic promise + logo
- 5.5-9.5s prompt invocation
- 9.5-14.5s generated result field
- 14.5-17.0s local/platform capability
- 17.0-20.5s connect ecosystem
- 20.5-24.5s model/capability ring
- 24.5-30.5s export burst
- 30.5-36.0s final claim

Use 6-8 scene modules.

## Extended: 45-60s

Use when a product needs more explanation, but keep the same visual grammar.

- 0.0-6.0s kinetic promise
- 6.0-12.0s prompt invocation and first result
- 12.0-20.0s product workflow
- 20.0-28.0s integrations/models/capabilities
- 28.0-38.0s real outputs and use cases
- 38.0-48.0s proof, platform support, or open-source angle
- 48.0-60.0s final claim and logo

Avoid turning this into a slide deck. Keep objects moving and transforming.
