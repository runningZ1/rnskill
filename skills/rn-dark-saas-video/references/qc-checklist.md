# QC Checklist

## Style Checks

- black spatial stage is present
- bottom purple horizon glow is subtle
- no persistent horizontal neon-line background dominates the video
- kinetic typography appears at opening or ending, preferably both
- gradient CTA exists and visibly triggers a transformation
- UI objects feel summoned from darkness
- primary subjects are large enough in hero frames
- no unexplained solid-color full-screen flash

## Motion Checks

- every scene has a readable hero frame
- major transitions use velocity, smear, blur, scale, or wipe
- text transformations happen at intentional beats
- cursor clicks have feedback when used
- rings/arcs/floating cards are deterministic and seek-safe

## Production Checks

- HyperFrames lint/validate/inspect pass when HyperFrames is used
- final MP4 exists in workbench and archive
- media specs are confirmed
- contact sheet covers all main scenes
- project note names the duration preset and selected scene modules

## Boundary Check

If the user asked for exact replication or says "not aligned", stop using this
skill as the primary route and use `rn-replica-qc`.
